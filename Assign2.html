What is Bootstrap?

The world's most popular framework for building responsive, mobile-first sites.

Definition üìù

Bootstrap is a free, open-source **front-end development framework** that provides HTML, CSS, and (optionally) JavaScript templates for common user interface components, typography, forms, buttons, tables, navigation, and more.

It's built with a **mobile-first** philosophy, meaning it ensures websites look and function great on smaller screens before scaling up to desktops. Its primary goal is to make web development faster and easier by providing a consistent, ready-to-use library of styles and components.

The core of Bootstrap is its responsive **Grid System**, which allows developers to easily lay out page content using rows and a 12-column structure.
